# stock-analysis
This project focuses on analyzing historical stock market data to uncover trading patterns, evaluate company performance, and forecast future stock prices using statistical and machine learning techniques.
The analysis helps investors make data-driven decisions by identifying reliable stocks, understanding market behavior, and predicting future trends.

#objective
Analyze historical stock market data.
Identify profitable trading patterns and trends.
Evaluate company performance based on stock movements.
Predict future stock prices using machine learning models.
Discover key factors influencing stock price fluctuations.

#Tools used
Python (Pandas, Seaborn, Matplotlib),Exception Handling,logging.
SQL for data management

#Outcome
Technical analysis reports and visualizations. 
Predictive models for price forecasting
